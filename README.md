# Отчет по анализу регульярного выражения

## Имется академическое регулярное выражение:
(ab|bc|bbbb|bbb)*bb(a|b)(a|b)(cac|accc|bca)*
соответсвенно алфавит описыаюзий это вывражение {ε, a, b, c}
## Построение ε-НКА:
Для начала по заданному регулярному выражению построим ε-НКА, постепеноо расписывая каждый переход:
<img width="2323" height="417" alt="image" src="https://github.com/user-attachments/assets/78b467c3-aeb9-4c15-8365-e7d0dc22f7c8" />

чтобы убедится в правильности и соотествии данного НКА и нашего выражение достатно увидеть соотестия состояний и переходов из них с регулярным выражением:<br>
[0]-[18] = ε = (ab|bc|bbbb|bbb)*  которое повторялось 0 раз, т.е превращается в ε<br>
[0]-[1] = ε = вхожденеие в один из цикловв повтора (ab|bc|bbbb|bbb)*<br>
далее внутри (ab|bc|bbbb|bbb) есть 4 развитления:<br>
[2]-[3]-[4]=ab<br>
[5]-[6]-[7]=bc<br>
[8]-[9]-[10]-[11]-[12]=bbbb<br>
[13]-[14]-[15]-[16]=bbb<br>
[17]-[0]= ε = перезапуск цикла<br>
Все это описывает звезду Клини для (ab|bc|bbbb|bbb)*.<br>
Дальше идет обязательноя часть слова bb(a|b)(a|b), описываемая состояниями с [18] по [30],<br>
и аналогично первой части описывваеются звезда Клини для части выражения (cac|accc|bca)*  состояними с [30] по [46].<br>

## Построение НКА:
На основе данного регулярного выражения рассмотрим его факторизацию (ab|bc|bbbb|bbb)*bb(a|b)(a|b)(cac|accc|bca)*,<br>
рассмотрим слово *w* которе пораждается данным выражением, причем для него выполняется:<br>
*w* = *w1* *w2* *w3*, где  <br>
*w1* - описывает (ab|bc|bbbb|bbb)* <br>
*w2* - описывает bb(a|b)(a|b) <br>
*w3* - описывает (cac|accc|bca)* <br>
Тогда заметим, что  *w1* и *w3* могут принимать значения ε, а *w2* будет обязательной частью слова.
ниже приведен НКА потсроенный по *w* = *w1* *w2* *w3*:
<img width="1126" height="631" alt="image" src="https://github.com/user-attachments/assets/5d7446de-ae6e-4c42-b576-049ef53cd8bd" />

Где для него ссостояния и перерходы между ними описывают:<br>
q0- стартовое состяние(вхождение)<br>
q0, q1, ... q4 - *w1* <br>
q0, q5, q6 ... q8 - *w2* <br>
q8, q9, ... q15 - *w3* <br>
Поэтому постоенный НКА послоностью соответсвует нашему регулярному выражению.


## Построение возможно малого НКА:
Но по построенному НКА можно сократить за счет Факторизацией с перекрытием, ведь правая чать *w1*, также может являться левой чатью слова *w2*,<br>
за счет этого былпостроен возможно малый НКА:

В итоге получаем 13 конечнынх сосотояний, с уже размытым соблюжением *w* = *w1* *w2* *w3*

## Построение ПКА:
На этой же стуктуре слова  *w* = *w1* *w2* *w3*, можем построить ПКА где из входящего состояния необоходимо выполнение следующих условий:<br>
1. Слово начинается на (ab|bc|bbbb|bbb)*
2. Слово содержит bb(a|b)(a|b)
3. Слово кончается на (cac|accc|bca)*
тоесть слово распознается автоматом если 1, 2 и 3 условие выполняется.
Вот сам ПКА:
<img width="1280" height="1002" alt="image" src="https://github.com/user-attachments/assets/b0498322-6431-4045-8ec3-527f1ad8c7a4" />

тут вырбранное слово *w* подается в виде 3 конечных сосотояний *w* = *q0 k4 c0*,
т.е. слово доллжно разбиаться на 3 состояния, и поэтом каждое из соостояний проверяется с коньюнкцией.

##Расширенное регулярное выражение:
по имеющемуся регулярному выражению надо построить расширенное регулярное выражение, распознающее тот же язык.<br>
